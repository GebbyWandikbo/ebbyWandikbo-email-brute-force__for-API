Script ini digunakan untuk menguji daftar email terhadap endpoint API tertentu untuk memeriksa validitasnya. Anda perlu menyediakan file wordlist yang berisi daftar email yang akan diuji, memastikan file tersebut berada di lokasi yang sesuai dengan konfigurasi. Jalankan script menggunakan Python, dan script akan membaca setiap email dalam wordlist satu per satu. Saat memproses, script akan mencetak status setiap email (valid, invalid, atau error) di terminal dengan warna yang berbeda untuk mempermudah pemantauan. Jika email valid ditemukan, script akan secara otomatis menyimpannya ke dalam file output di folder yang sama dengan lokasi script. Tidak ada threading, sehingga setiap email diproses secara berurutan.
